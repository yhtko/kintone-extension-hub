PlugBits Launcher for kintone
kintone作業を高速化するランチャー

概要

PlugBits Launcher は
kintoneの操作を高速化する Chrome拡張です。

サイドパネルランチャーと ExcelライクなUIにより
日常作業をより快適にします。

主な機能
サイドパネルランチャー

kintone画面から直接アクセスできる作業ランチャー。

WatchList（ビュー件数確認）

Record Pin（レコード固定）

Recent Records（履歴）

画面遷移を減らし、作業を高速化します。

Excel Overlay

kintoneの一覧・詳細を
ExcelのようなUIで表示できます。
