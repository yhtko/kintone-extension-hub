Privacy Policy（日本語）
PlugBits Launcher for kintone プライバシーポリシー

本拡張機能「PlugBits Launcher for kintone」（以下「本拡張」）は、ユーザーのプライバシーを尊重し、個人情報の収集・利用・共有について以下の方針に従います。

1. 収集する情報

本拡張は、ユーザーが設定した以下の情報を Chrome 拡張ストレージに保存します。

ピン留めしたレコード情報

お気に入り（ショートカット）設定

最近アクセスしたレコード情報

表示設定（列幅、UI設定など）

本拡張の各種設定

これらの情報は、拡張機能の機能提供のためにのみ使用されます。

2. 保存場所

上記の情報は、以下の Chrome 拡張ストレージに保存されます。

chrome.storage.local

chrome.storage.sync

chrome.storage.sync を利用する場合、Chrome の同期機能によりユーザーのブラウザ間で設定が同期される場合があります。

3. 通信について

本拡張は、kintone 上での機能提供のために、ユーザーが利用している kintone 環境に対して API 通信を行います。

通信対象は以下のドメインに限定されています。

*.kintone.com

*.cybozu.com

*.kintone-dev.com

本拡張は、これら以外の第三者サーバーへユーザーデータを送信することはありません。

4. 個人情報の収集

本拡張は、以下の情報を収集しません。

個人を特定できる情報

ブラウザの閲覧履歴

Cookie の読み取り

外部トラッキング情報

5. 第三者への提供

本拡張は、ユーザーのデータを以下の用途で使用しません。

第三者への販売

第三者への共有

広告またはマーケティング目的の利用

6. 外部スクリプト

本拡張は、外部 CDN や外部スクリプトを読み込むことはありません。

7. プライバシーポリシーの変更

本ポリシーは必要に応じて更新される場合があります。

Privacy Policy (English)
Privacy Policy for PlugBits Launcher for kintone

The Chrome extension "PlugBits Launcher for kintone" respects user privacy and follows the policies described below regarding the collection and use of information.

1. Information Stored

The extension stores the following information in Chrome Extension Storage:

Pinned record information

Favorite / shortcut settings

Recently accessed records

Display preferences (such as column width and UI settings)

Extension configuration settings

This information is used solely to provide the extension's functionality.

2. Storage Location

The data listed above is stored using Chrome Extension Storage:

chrome.storage.local

chrome.storage.sync

If chrome.storage.sync is used, the settings may be synchronized across the user's Chrome browsers.

3. Network Communication

To provide its features, the extension may communicate with the user's kintone environment using the kintone REST API.

Communication is limited to the following domains:

*.kintone.com

*.cybozu.com

*.kintone-dev.com

The extension does not send user data to any third-party servers.

4. Personal Data Collection

The extension does not collect:

Personally identifiable information

Browsing history

Cookies

Tracking or analytics data

5. Data Sharing

User data is not:

Sold

Shared with third parties

Used for advertising or marketing

6. External Scripts

The extension does not load external scripts or third-party CDNs.

7. Policy Updates

This privacy policy may be updated when necessary.
