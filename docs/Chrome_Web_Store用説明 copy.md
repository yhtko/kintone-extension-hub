PlugBits Launcher for kintone

PlugBits Launcher for kintone improves navigation and productivity in kintone by providing quick access to frequently used apps, views, and records.

The extension adds convenient shortcuts, pinning features, and a side panel launcher to help users navigate large kintone environments more efficiently.

Main Features
Quick Launcher

Access frequently used apps and views instantly.

Record Pinning

Pin important records and open them quickly from the launcher.

Favorites & Shortcuts

Save commonly used apps, views, and create pages as shortcuts.

Side Panel Navigation

Open the launcher in a Chrome side panel for faster navigation while working in kintone.

Record Count Display

Display record counts for views to understand workload and data volume.

Spreadsheet View (View Mode)

Supports viewing records in spreadsheet-style layout for easier browsing.

Designed for kintone users

This extension is designed specifically for organizations that use kintone extensively and need faster navigation between apps and views.

It helps reduce repetitive navigation and improves productivity when working with multiple apps.

Data & Privacy

The extension stores user settings using Chrome Extension Storage.

Communication is limited to the user's kintone environment.

No data is sent to external third-party servers.

No tracking or analytics are used.

See the Privacy Policy for more details.
